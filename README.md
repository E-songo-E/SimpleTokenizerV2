# SimpleTokenizerV2


This is a custom Byte Pair Encoding (BPE) tokenizer built using the Hugging Face `tokenizers` library. 
It is trained on the Cool Textz English dataset [the Cool Textz English dataset](https://huggingface.co/datasets/cosodavvero/cool-textz-english)..

## Features
- BPE-based tokenization
- Encoding and decoding support
- Customizable vocab size and special tokens

## How to Use
